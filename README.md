# 자동차 경주 게임

## 구현 필요 기능 리스트

* 이름의 목록을 쉽표로 구분받아 입력받는 UI
* 이름이 5자 이하인지 확인하고 예외를 던지는 메서드
* 시뮬레이션 횟수 입력받는 UI
* 랜덤 값 생성기
* 랜덤 값이 4 이상여부 판단하는 함수
* 전진, 멈춤 수행하는 함수
* 각 자동차에 이동 시뮬레이션을 수행하는 함수
* 1회 시뮬레이션마다 현황을 출력하는 함수
* 이동 결과로 우승자 찾아내는 함수
* 우승자 출력 함수
* 랜덤 값이 0 ~ 9 사이 값인지 판단하는 테스트
* 경주가 완료여부 판단함수
* 최종 우승자 판단

## 클래스별 책임

* Car: 이름을 가진 자동차 생성 및 전진 등 자동차 관련 작업.  
* GameManager: 게임 전반 관련 작업 수행. (자동차 시뮬레이션)  
* Input: 사용자 입력 관련 작업 수행.
* Output: 사용자 출력 관련 작업 수행.  
* RandomNumber: 랜덤 숫자 생성 작업 수행.
